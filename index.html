Tag h1

É usada para identificar o título mais importante da página, como o nome do artigo ou o título da seção mais importante. Por padrão, o texto dentro da tag <h1> é exibido em uma fonte maior e mais negrito do que o texto normal.

Índice

Tag h2

É usada para definir um subtítulo ou uma seção secundária dentro de um documento HTML. É uma das tags de cabeçalho, com <h1> porém 1 nível abaixo. A <h2> pode ser usada para subdividir seções de uma página em partes mais manejáveis e organizadas.

Índice

Tag h3

A tag <h3> em HTML é outra das tags de cabeçalho usadas para organizar o conteúdo de uma página da web. Normalmente usada para criar subtítulos ou seções que são subordinadas aos títulos definidos com <h1> e <h2>. Ela ajuda a subdividir o conteúdo em partes menores e mais gerenciáveis, fornecendo uma estrutura hierárquica clara.

Índice

Tag h4

A tag <h4> no HTML é uma das tags de cabeçalho usadas para definir a importância e hierarquia dos títulos em uma página web.

Índice

Tag h5

A tag <h5> no HTML é usada para definir um cabeçalho de nível 5. Ela é menos importante hierarquicamente comparada às tags <h1> a <h4>, mas ainda pode ser usada para criar uma estrutura lógica de títulos e subtítulos na sua página web

Índice

Tag h6

A tag <h6> é a menor e menos importante na hierarquia das tags de cabeçalho em HTML. Ela é usada para definir um cabeçalho de nível 6, o que indica que é um subtítulo ou uma seção ainda mais específica dentro da estrutura de conteúdo da sua página web.

Índice

Tag p

É usada para definir um parágrafo de texto. É um dos elementos fundamentais e comuns em HTML. Cada parágrafo definido com a tag <p> será exibido em uma nova linha com um espaço de margem acima e abaixo por padrão.

Índice

Tag mark

É usada para destacar ou marcar partes do texto que são relevantes ou importantes dentro do conteúdo. Normalmente, o texto dentro de uma tag <mark> é exibido com um fundo amarelo ou outra cor de destaque, para chamar a atenção do usuário para aquela parte específica do texto.

Índice

Tag small

É usada para diminuir o tamanho do texto em relação ao texto ao redor. É frequentemente utilizada para indicar texto adicional, como notas de rodapé, informações secundárias ou detalhes que não precisam de destaque principal.

Índice

Tag i

É usada para representar texto que deve ser exibido em itálico. Originalmente, a tag <i> tinha um significado semântico para texto em itálico, mas em HTML5, seu uso semântico foi reduzido. A tag <i> é mais adequada para representar texto com uma aparência diferente, como palavras estrangeiras, termos técnicos, ou nomes de obras, sem carregar um significado semântico específico.

Índice

Tag u

É usada para sublinhar texto. Ela foi originalmente usada para marcar texto como sublinhado, mas sua utilização tem se tornado menos comum em HTML moderno devido a considerações de acessibilidade e semântica

Índice

Tag strong

É usada para indicar que o texto dentro dela deve ser exibido com um peso de fonte maior, geralmente em negrito. Além de estilizar o texto em negrito, a tag <strong> tem um significado semântico. Ela indica que o texto é de maior importância ou que deve ser enfatizado.

Índice

Tag ol

É usada para criar uma lista ordenada, onde cada item da lista é numerado automaticamente pelo navegador. Listas ordenadas são úteis quando a ordem dos itens é importante, como em passos de instruções, classificações ou qualquer outro tipo de lista onde a sequência é significativa.

Índice

Tag ul

É usada para criar uma lista não ordenada, onde os itens são exibidos com marcadores. Diferente das listas ordenadas, que utilizam números ou letras para indicar a ordem dos itens, as listas não ordenadas utilizam marcadores, como pontos ou círculos.

Índice

Tag li

É usada para definir um item em uma lista ordenada (<ol>) ou não ordenada (<ul>). Cada item da lista é envolvido por uma tag <li>, e essas tags são colocadas dentro da tag <ul> ou <ol>.

Índice

Tag a

É usada para criar links, permitindo a navegação entre diferentes páginas ou recursos na web. É uma das tags mais importantes para a criação de hyperlinks e navegação em sites.

Índice

Tag hr

É usada para criar uma linha horizontal, que geralmente é utilizada para separar seções de conteúdo em uma página.

Índice

Tag sub

É usada para exibir texto em subscrito. O texto dentro da tag <sub> é exibido em uma posição inferior em relação ao texto padrão, e geralmente é menor. Essa tag é frequentemente usada em fórmulas matemáticas, notações químicas ou outras situações onde um texto deve aparecer em uma posição abaixo da linha de base

Índice

Tag blockquote

É usada para marcar um bloco de citação de uma fonte externa. É comumente utilizada para destacar citações longas ou trechos de texto que são retirados de outra fonte, como livros, artigos ou outros sites.
